# 🌟 Suresh Nepali | Personal Portfolio

Welcome to my **personal portfolio website**!  
This project showcases my **skills, projects, certificates, and services** as a Full Stack Web Developer and Tech Enthusiast.  
The portfolio is **modern, responsive, and interactive**, built with HTML5, CSS3, and JavaScript.  

---

## 🚀 Features

- 🏠 **Hero Section** with dynamic animated roles  
- 👤 **About Me** section highlighting background and interests  
- 💻 **Skills** in a responsive grid layout  
- 🖼️ **Projects / Work** showcased with cards and live links  
- 📜 **Certificates** in grid layout  
- 🎨 **Services Offered** in cards  
- ✉️ **Contact Section** with email and social links  
- 📱 **Responsive design** for desktop and mobile  
- ✨ **Smooth animations** with hover and transitions  


## Technologies Used

- **HTML5** – Semantic structure of the website  
- **CSS3** – Styling, Flexbox, Grid, transitions, responsive design  
- **JavaScript** – Animations for dynamic text and interactivity  
- **Optional Tools** – Python/Flask (for backend projects), Git & GitHub
